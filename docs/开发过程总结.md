# 麦田智囊开发过程总结

## 项目概述

**项目名称**：麦田智囊 - 基于LangChain与自进化RAG的乡村教育Agent系统

**项目目标**：为乡村教师提供专属AI教研搭档，实现教学经验数字化传承

---

## 开发时间线

### 第一阶段：问题诊断与修复

**问题**：
- 运行demo时出现 `ModuleNotFoundError: No module named 'langchain.prompts'`

**原因分析**：
- LangChain 1.2.15版本中，prompts模块已移至langchain_core
- paddleocr无法在Python 3.14环境中安装

**解决方案**：
1. 修改导入语句：`from langchain.prompts` → `from langchain_core.prompts`
2. 替换OCR方案：paddleocr → easyocr

**修改的文件**：
- `demo/agents/quick_lesson_prep.py`
- `demo/agents/wisdom_transfer.py`

## 技术决策记录

### 1. OCR方案选择
- **原计划**：PaddleOCR
- **实际**：EasyOCR
- **原因**：PaddleOCR不支持Python 3.14，EasyOCR兼容性好
- **影响**：识别效果略有差异，但用户体验一致

### 2. 导入路径更新
- **原因**：LangChain版本迭代，prompts模块迁移到langchain_core
- **影响**：需要修改所有导入语句

### 3. 项目结构
- **demo/** - 演示版本，包含已完成功能的最小化实现
- **maitian_agent/** - 完整项目框架，包含所有模块的完整实现

## 部署说明

### Demo运行
```bash
pip install -r requirements.txt
streamlit run demo/app.py
```

### 完整框架运行
```bash
pip install -r requirements.txt
streamlit run maitian_agent/frontend/streamlit_app.py
# 或
uvicorn maitian_agent.api.routes:app --host 0.0.0.0 --port 8000
```

### Docker部署
```bash
docker-compose -f docker/docker-compose.yml up -d
```
